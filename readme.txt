Create your own community with the best social networking plugin for WordPress. Beautiful user profiles, activity stream, notifications, member listing, blog posts integration with comments, reactions, hashtags and many more.

== Description ==

[youtube https://www.youtube.com/watch?v=xu9XkT3VDMo]

Allows to create account on WordPress site using existing social network accounts. One-click login to your website using different Social Networking accounts or single sign on features using pre-configured application. Allow your users to link their WordPress account to social login account.

Supported providers:
* Facebook
* Google
* Instagram
* Twitter
* Reddit

== Other Features ==
* One click registration through any of the supported providers.
* Possibility to disconnect Social Account and just stay with a WordPress login / password.
* Importing avatars, fist name, last name, birthdate information - depending on what information is available through a provider API.
* Customizable provider icons.
* Quick to setup and easy to use.
 
== Getting Started ==
 
To get a social network on your WordPress site up and running, follow these steps:
 
* Login to your WordPress dashboard.
* Choose "Plugins".
* Click "Add new".
* Search for "Social Login & Invitations by PeepSo".
* Click "Install".
* Once installed, activate and follow instructions for setting up each provider.

== Installation ==

= From your WordPress dashboard =

1. Visit "Plugins > Add New"
2. Search for "Social Login & Invitations by PeepSo"
3. Press install
4. Activate the plugin from your Plugins page.


= From WordPress.org =

1. Download Social Login & Invitations by PeepSo
2. Navigate to the Admin area of your site.
3. Go to Plugins > "Add New".
4. Select "Upload Plugin".
5. Select the ZIP installation file of Social Login & Invitations by PeepSo.
6. Activate Social Login & Invitations by PeepSo from your Plugins page.
